<task type="auto">
  <name>Task 2: FFmpeg service wrappers — normalize, probe, and thumbnail</name>
  <files>

  </files>
  <action>
    **`packages/backend/src/services/ffmpeg.ts`:**
    Create typed Promise wrappers around `child_process.spawn` for FFmpeg and FFprobe.

    Constants:
    - `FFMPEG = '/opt/homebrew/bin/ffmpeg'`
    - `FFPROBE = '/opt/homebrew/bin/ffprobe'`

    At module load time, verify both binaries exist using `execFileSync('which', ['ffmpeg'])` wrapped in try/catch. If not found, log a clear error: "FFmpeg not found. Install with: brew install ffmpeg" and throw.

    **`normalizeVideo(inputPath: string, outputPath: string, onProgress?: (percent: number) => void): Promise<void>`**
    - Spawn FFmpeg with args: `-i inputPath -c:v libx264 -preset fast -crf 18 -r 30 -c:a aac -b:a 192k -movflags +faststart -y outputPath`
    - Also pass `-progress pipe:1` to get machine-readable progress on stdout
    - Parse `out_time_ms` from stdout lines to compute progress percentage (requires knowing total duration first — accept optional `durationMs` param, or probe before normalize)
    - Collect stderr for error reporting
    - On exit code 0 resolve; on non-zero reject with last 500 chars of stderr
    - Call `onProgress` callback with 0-100 percent as normalization progresses
    - IMPORTANT: Never use `exec` — it buffers all output in memory. Always use `spawn`.

    **`probeVideo(filePath: string): Promise<VideoMetadata>`**
    - Spawn FFprobe with args: `-v quiet -print_format json -show_streams -show_format filePath`
    - Parse stdout JSON, find the video stream (`codec_type === 'video'`), extract:
      - `duration`: from `format.duration` (parseFloat, seconds)
      - `fps`: from video stream `r_frame_rate` (parse "num/den" fraction, round)
      - `width`: from video stream `width`
      - `height`: from video stream `height`
      - `codec`: from video stream `codec_name`
    - Return typed `VideoMetadata` object (imported from `@eigen/shared-types`)

    **`packages/backend/src/services/thumbnail.ts`:**
    **`extractThumbnail(inputPath: string, outputPath: string): Promise<void>`**
    - Spawn FFmpeg with args: `-ss 1 -i inputPath -vframes 1 -q:v 2 -y outputPath`
    - On exit code 0 resolve; on non-zero reject
    - The `-ss` before `-i` is important for fast seeking

    All functions should be exported individually (named exports). Import Job/VideoMetadata from `@eigen/shared-types`.
  </action>
