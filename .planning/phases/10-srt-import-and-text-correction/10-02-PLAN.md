**1. Create `packages/frontend/src/components/TextEditor/SrtDiffView.tsx`:**

```tsx
import { diffWords } from 'diff'
import type { AlignedPhrase } from '../../lib/srtAlignment.ts'
import './SrtDiffView.css'

interface SrtDiffViewProps {
  alignedPhrases: AlignedPhrase[]
  onAccept: (alignedIndex: number) => void
  onReject: (alignedIndex: number) => void
  onDismiss: () => void
}

export function SrtDiffView({ alignedPhrases, onAccept, onReject, onDismiss }: SrtDiffViewProps)
```

Component structure:
- Outer div `className="srt-diff"`
- **Header:** div `className="srt-diff__header"`
  - h3 `className="srt-diff__title"` text: "SRT Diff Review"
  - p `className="srt-diff__subtitle"` text: `{N} phrase{s} matched — accept or reject each correction` (use `alignedPhrases.length`, pluralize: "phrase" when 1, "phrases" otherwise)
  - button `className="srt-diff__dismiss"` `aria-label="Discard SRT diff"` content: x glyph (unicode `\u00D7`)
- **Empty state** (when `alignedPhrases.length === 0`): div `className="srt-diff__empty"` with two lines:
  - "No matching phrases found."
  - "Check that the SRT timestamps overlap with the transcript."
- **Column headers** (when phrases exist): div `className="srt-diff__columns"` sticky
  - span "Whisper" / span "SRT" / span (empty, for buttons column)
- **Scrollable row list:** div `className="srt-diff__scroll"`
  - For each aligned phrase (by index in alignedPhrases array):
    - div `className="srt-diff__row"` with `key={aligned.phraseIndex}`
    - Line number: span `className="srt-diff__line-num"` showing `aligned.phraseIndex + 1`
    - Whisper column: div `className="srt-diff__col srt-diff__col--whisper"` — render inline diff of whisper text (removed words get `<s className="srt-diff__removed">`)
    - SRT column: div `className="srt-diff__col srt-diff__col--srt"` — render inline diff of SRT text (added words get `<mark className="srt-diff__added">`)
    - Actions column: div `className="srt-diff__actions"`
      - button `className="srt-diff__accept"` text: "Accept Correction" — calls `onAccept(alignedIndex)`
      - button `className="srt-diff__reject"` text: "Reject Change" — calls `onReject(alignedIndex)`

**Inline diff rendering** — helper function `renderDiff(whisperText, srtText, side: 'whisper' | 'srt')`:
- Call `diffWords(whisperText, srtText)`
- Return array of React elements:
  - `part.added && side === 'srt'` -> `<mark className="srt-diff__added">{part.value}</mark>`
  - `part.removed && side === 'whisper'` -> `<s className="srt-diff__removed">{part.value}</s>`
  - `part.added && side === 'whisper'` -> skip (don't show additions in whisper column)
  - `part.removed && side === 'srt'` -> skip (don't show removals in srt column)
  - Otherwise: `<span>{part.value}</span>`
