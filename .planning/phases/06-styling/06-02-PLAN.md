    Create packages/frontend/src/components/StylePanel/ directory.

    1. Create StylePanel.tsx with the following controls, each bound to useSubtitleStore's setStyle action:

       a. **Font selector** — `<select>` element populated from FONT_NAMES (imported from '@eigen/remotion-composition'). Each `<option>` value = getFontFamily(name), label = font name. Style the select element with the selected font family for preview. Current value reads from `useSubtitleStore((s) => s.style.fontFamily)`.

       b. **Font size slider** — `<input type="range">` with min=16, max=96, step=2. Display current value as "{fontSize}px". Reads from `useSubtitleStore((s) => s.style.fontSize)`.

       c. **Highlight color picker** — HexColorPicker + HexColorInput from 'react-colorful'. Reads from `useSubtitleStore((s) => s.style.highlightColor)`. Label: "Highlight color (active word)".

       d. **Base color picker** — Same pattern as highlight color but for `baseColor`. Label: "Base color (inactive words)".

       e. **Stroke width slider** — `<input type="range">` with min=0, max=4, step=0.5. Display "{strokeWidth}px". Show the stroke color picker only when strokeWidth > 0.

       f. **Stroke color picker** — HexColorPicker + HexColorInput, shown conditionally when strokeWidth > 0. Reads from `useSubtitleStore((s) => s.style.strokeColor)`.

       g. **Vertical position slider** — `<input type="range">` with min=5, max=95, step=1. Display "{verticalPosition}% from top". Reads from `useSubtitleStore((s) => s.style.verticalPosition)`.

    Each control should be wrapped in a labeled section with a descriptive label. Use individual Zustand selectors for each field (not the whole style object) to minimize re-renders.

    2. Create StylePanel.css following the project's co-located CSS pattern (dark theme, matching existing component styles like TranscriptEditor.css):

       - `.style-panel__section` — grouped control with label, margin-bottom for spacing
       - `.style-panel__label` — section heading, muted color, small caps or similar
       - `.style-panel__slider` — consistent slider styling
       - `.style-panel__color-control` — layout for color picker + hex input
       - `.style-panel__select` — dropdown styling matching dark theme
       - Color pickers should be reasonably sized (~200px wide, not full width) to avoid dominating the panel
       - Override react-colorful's default styles to match the dark theme background
