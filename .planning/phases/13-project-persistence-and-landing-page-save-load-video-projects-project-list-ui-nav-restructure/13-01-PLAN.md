1. Create `packages/backend/src/routes/projects.ts`:

- Import `fp`, `FastifyInstance`, `crypto`, `path`, `{ rm } from 'node:fs/promises'`, `{ createReadStream, existsSync } from 'node:fs'`, `{ DATA_ROOT } from '../index.ts'`

- Define `ProjectRow` interface: `{ id: string, job_id: string, name: string, state_json: string | null, duration: number | null, created_at: number, updated_at: number }`

- `rowToProject(row: ProjectRow)` mapper: returns `{ id: row.id, jobId: row.job_id, name: row.name, stateJson: row.state_json, duration: row.duration, createdAt: row.created_at, updatedAt: row.updated_at }`

- Prepared statements (all using `fastify.projectsDb.prepare`):
  - `getAllStmt`: `SELECT id, job_id, name, duration, created_at, updated_at FROM projects ORDER BY updated_at DESC` (omit state_json for list — it can be large)
  - `getByIdStmt`: `SELECT * FROM projects WHERE id = ?`
  - `insertStmt`: `INSERT INTO projects (id, job_id, name, state_json, duration, created_at, updated_at) VALUES (?, ?, ?, ?, ?, ?, ?)`
  - `updateNameStmt`: `UPDATE projects SET name = ?, updated_at = ? WHERE id = ?`
  - `updateStateStmt`: `UPDATE projects SET state_json = ?, updated_at = ? WHERE id = ?`
  - `deleteStmt`: `DELETE FROM projects WHERE id = ?`
  - `countByJobIdStmt`: `SELECT COUNT(*) as cnt FROM projects WHERE job_id = ?`

- Routes:

**GET /api/projects** — `getAllStmt.all()` mapped through `rowToProject`. Returns array (no stateJson in list response).

**POST /api/projects** — Body: `{ jobId: string, name: string, duration?: number }`. Generate `id = crypto.randomUUID()`, `now = Date.now()`. `insertStmt.run(id, jobId, name, null, duration ?? null, now, now)`. Return 201 with `{ id, jobId, name, duration, createdAt: now, updatedAt: now }`.

**GET /api/projects/:id** — `getByIdStmt.get(id)`. If not found, 404 `{ error: 'Project not found' }`. Return full `rowToProject(row)` including stateJson.

**PUT /api/projects/:id** — Body: `{ name?: string, stateJson?: string }`. If name provided: `updateNameStmt.run(name, Date.now(), id)`. If stateJson provided: `updateStateStmt.run(stateJson, Date.now(), id)`. If both: run both statements. If project not found (check first with getByIdStmt), return 404. Return 200 with updated project via `getByIdStmt.get(id)` mapped.

**DELETE /api/projects/:id** — Get row first via getByIdStmt. If not found, 404. Check `countByJobIdStmt.get(row.job_id)` — if `cnt > 1` (other projects share this jobId due to Duplicate), only delete the project record, NOT the files. If `cnt === 1`, also delete files: `rm(path.join(DATA_ROOT, row.job_id), { recursive: true, force: true }).catch(() => {})`. `deleteStmt.run(id)`. Return 204.

**POST /api/projects/:id/duplicate** — Get row via getByIdStmt. If not found, 404. New `id = crypto.randomUUID()`, `now = Date.now()`, `name = row.name + ' (copy)'`. `insertStmt.run(id, row.job_id, name, row.state_json, row.duration, now, now)`. Return 201 with new project.

**GET /api/jobs/:jobId/thumbnail** — `const thumbPath = path.join(DATA_ROOT, jobId, 'thumbnail.jpg')`. If `!existsSync(thumbPath)`, return 404 `{ error: 'Thumbnail not found' }`. Otherwise `reply.type('image/jpeg').send(createReadStream(thumbPath))`.

- `export default fp(projectsRoutes)`
