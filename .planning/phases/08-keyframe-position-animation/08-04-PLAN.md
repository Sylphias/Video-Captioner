    Create `KeyframeTrackRow.tsx` — a single horizontal row representing one keyframeable property.

    **Props interface:**
    ```typescript
    import type { KeyframeableProperty, KeyframeTrack, KeyframeEasing } from '@eigen/shared-types'

    interface KeyframeTrackRowProps {
      property: KeyframeableProperty
      track: KeyframeTrack | undefined  // undefined = no keyframes for this property yet
      playheadProgress: number  // 0-1, current playhead position
      selectedKeyframeIndex: number | null
      isSelected: boolean  // is this property row selected
      onSelectProperty: () => void
      onAddKeyframe: (time: number, value: number) => void
      onRemoveKeyframe: (index: number) => void
      onMoveKeyframe: (index: number, newTime: number) => void
      onSelectKeyframe: (index: number | null) => void
      onSetEasing: (segmentIndex: number, easing: KeyframeEasing) => void
    }
    ```

    **Layout:** Each row is a horizontal bar, ~32px tall.
    ```
    [Property Label (60px)] [─────────── Track Area (flex: 1) ───────────]
    ```

    **Property label:** Left-side fixed-width label showing the property name: "X", "Y", "Scale", "Rotation", "Opacity". Background slightly different for selected row.

    **Track area:**
    - Full width represents 0% to 100% of phrase lifetime (left = 0, right = 1)
    - Background: `var(--surface-2)` with subtle horizontal gridlines at 25%, 50%, 75%
    - Keyframe diamonds: positioned at `left: ${kf.time * 100}%`, rendered as 8x8 rotated squares (CSS `transform: rotate(45deg)`). Color: `var(--accent)` when selected, `var(--text-muted)` otherwise.
    - Between consecutive keyframe diamonds, show a thin colored line segment. The line color varies by easing type (optional visual hint).
    - Double-click on empty track area at position X → `onAddKeyframe(x / trackWidth, defaultValue)` where defaultValue is: x=50, y=75, scale=1, rotation=0, opacity=1
    - Click on a diamond → `onSelectKeyframe(index)`
    - Drag a diamond horizontally → `onMoveKeyframe(index, newTime)` using pointer capture
    - Right-click (contextmenu) on a diamond → show a minimal context menu with "Delete keyframe" and "Set easing..." options. "Set easing..." opens the EasingPicker as a popover near the clicked segment.
    - Delete key when a keyframe is selected → `onRemoveKeyframe(index)`
    - Clicking on the segment between two keyframes opens EasingPicker as a positioned popover below the segment. The EasingPicker's onChange calls `onSetEasing(segmentIndex, easing)`.

    **Diamond drag:**
    - `pointerdown` on diamond: setPointerCapture, set dragging state
    - `pointermove`: compute new time = (clientX - trackAreaRect.left) / trackAreaRect.width, clamp to [0, 1]
    - `pointerup`: end drag, call `onMoveKeyframe`

    **CSS (KeyframeTrackRow.css):**
    - `.keyframe-track-row`: display flex, height 32px, align-items center, border-bottom 1px solid var(--border)
    - `.keyframe-track-row--selected`: background rgba(var(--accent-rgb), 0.05)
    - `.keyframe-track-row__label`: width 60px, padding 0 8px, font-size 11px, color var(--text-muted), text-transform uppercase, font-weight 600, cursor pointer, flex-shrink 0, user-select none
    - `.keyframe-track-row__track`: position relative, flex 1, height 100%, overflow hidden
    - `.keyframe-track-row__diamond`: position absolute, width 8px, height 8px, transform translate(-50%, -50%) rotate(45deg), top 50%, cursor grab, z-index 2, border 1px solid var(--border)
    - `.keyframe-track-row__diamond--selected`: background var(--accent), border-color var(--accent)
    - `.keyframe-track-row__diamond--default`: background var(--text-muted), border-color var(--text-muted)
    - `.keyframe-track-row__segment`: position absolute, top 50%, height 2px, transform translateY(-50%), background var(--text-muted), opacity 0.5, cursor pointer, z-index 1
    - `.keyframe-track-row__gridline`: position absolute, top 0, bottom 0, width 1px, background var(--border), opacity 0.3
