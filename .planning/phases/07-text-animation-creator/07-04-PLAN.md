    **Create packages/frontend/src/hooks/useDebounced.ts:**

    Generic debounce hook (from research Pattern 7):
    ```typescript
    import { useState, useEffect } from 'react'

    export function useDebounced<T>(value: T, delay: number): T {
      const [debounced, setDebounced] = useState<T>(value)
      useEffect(() => {
        const timer = setTimeout(() => setDebounced(value), delay)
        return () => clearTimeout(timer)
      }, [value, delay])
      return debounced
    }
    ```

    Visual layout: horizontal bar divided into three colored blocks:
    - Enter (green/teal): left block, width proportional to enterDuration/totalDuration
    - Active (gray/neutral): middle block, fills remaining space

    Interaction:
    - Click any block to select it (triggers `onSelectPhase`).
    - Selected block has a highlighted border/glow.
    - Drag handle between Enter and Active blocks: changes Enter duration. Drag left to shorten, right to lengthen. Clamped: min 0.05s, max totalDuration - exitDuration - 0.1s.
    - Drag handle between Active and Exit blocks: changes Exit duration. Same clamping logic.
    - Each block shows its label ("Enter", "Hold", "Exit") and duration in seconds.

    Drag implementation: `onMouseDown` on the handle -> track `clientX` delta -> convert to duration change via pixel-to-seconds ratio based on container width. Use `document.addEventListener('mousemove'/'mouseup')` pattern (matches existing resize handle in SubtitlesPage).
