Task 2 is a `checkpoint:decision` that requires manual execution:

### Steps for User

1. **Delete old .venv if it exists** (Mac deps won't work on Windows):
   ```
   rd /s /q .venv
   ```

2. **Install NeMo dependencies** (for spike):
   ```
   python -m venv .venv
   .venv\Scripts\pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu128
   .venv\Scripts\pip install triton-windows
   .venv\Scripts\pip install "nemo_toolkit[asr]"
   ```
   If any step fails with an unresolvable error, note which step and try WSL (see plan Task 2 Step 2b).

3. **Verify CUDA works**:
   ```
   .venv\Scripts\python -c "import torch; print(torch.cuda.is_available())"
   ```
   Should print `True`.

4. **Run the spike** (install + CUDA + model load):
   ```
   .venv\Scripts\python scripts/spike_parakeet.py
   ```
   Optionally with a test video: `.venv\Scripts\python scripts/spike_parakeet.py path/to/video.mp4`

5. **Report result and select Plan 02 path:**
   - **Option A** — Spike PASS (NeMo native, Parakeet CUDA, word timestamps work)
   - **Option B** — Spike WSL-ONLY (NeMo only works via WSL)
   - **Option C** — Spike FAIL/PARTIAL (fall back to WhisperX CUDA per D-02)
