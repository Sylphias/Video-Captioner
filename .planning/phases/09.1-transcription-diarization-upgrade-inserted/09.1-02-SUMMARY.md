## What was done

- Replaced Parakeet TDT with WhisperX large-v3 after benchmarking (better word timestamps, built-in alignment)
- Merged transcription + alignment + diarization into single `scripts/transcribe.py` pass (accepts hf_token + num_speakers)
- Kept `scripts/diarize.py` as standalone re-diarization for adjusting speaker count post-transcription
- Both scripts run on CUDA (RTX 4080) via WSL Python venv
- Backend passes HUGGINGFACE_TOKEN from .env to transcribe.py
- Frontend `useTranscribe` hook accepts optional numSpeakers; separate `useDiarize` hook for re-detection
- Output format matches existing schema: `{language, words: [{word, start, end, confidence, speaker}]}`
- Fixed: NaN speaker color for undiarized phrases, diarize.py num_speakers kwarg bug, soundfile over torchaudio for audio loading
