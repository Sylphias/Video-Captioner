    ```typescript
    mergePhrases: (indices: number[]) => void
    deletePhrases: (indices: number[]) => void
    duplicatePhrase: (phraseIndex: number) => void
    movePhraseUp: (phraseIndex: number) => void
    movePhraseDown: (phraseIndex: number) => void
    ```

    Implement `mergePhrases(indices)`:
    1. Guard: if !session or indices.length < 2, return state (no-op, NO pushUndo)
    2. pushUndo(state) — exactly once
    3. Sort indices ascending, deduplicate
    4. Collect all words from all selected phrase indices: `sorted.flatMap(i => state.session.phrases[i].words)`
    5. Build merged SessionPhrase: `{ words: mergedWords, isManualSplit: state.session.phrases[sorted[0]].isManualSplit, dominantSpeaker: computeDominantSpeaker(mergedWords) }`
    6. Filter out all selected phrases except first: `phrases.filter((_, i) => !sorted.includes(i) || i === sorted[0])`
    7. Set merged phrase at sorted[0]
    8. Rebuild manualSplitWordIndices: remove split indices that fall at boundaries between any merged phrases
    9. Return updated session

    Implement `deletePhrases(indices)`:
    1. Guard: if !session or indices.length === 0, return state
    2. Calculate total words that would remain after deletion — if 0, skip (refuse to delete all)
    3. pushUndo(state) — exactly once
    4. Sort indices descending (process from end to avoid index shifting)
    5. For each index: compute globalOffset, remove phrase words from flat words array, adjust manualSplitWordIndices (shift down by deleteCount for indices above)
    6. Rebuild phrases via `buildSessionPhrases(words, manualSplitWordIndices, state.maxWordsPerPhrase)`
    7. Return updated session

    Implement `duplicatePhrase(phraseIndex)`:
    1. Guard: if !session or invalid index, return state
    2. pushUndo(state)
    3. Clone the phrase's words with `structuredClone`
    4. Insert cloned words into flat words array at the position right after the source phrase's last word
    5. Add manual split index to isolate the clone as its own phrase
    6. Rebuild phrases via buildSessionPhrases

    Implement `movePhraseUp(phraseIndex)`:
    1. Guard: if !session or phraseIndex === 0, return state
    2. pushUndo(state)
    3. Swap phrases[phraseIndex] and phrases[phraseIndex-1] in the flat words array (swap the word blocks)
    4. Rebuild phrases via buildSessionPhrases

    Implement `movePhraseDown(phraseIndex)`:
    1. Guard: if !session or phraseIndex >= phrases.length - 1, return state
    2. pushUndo(state)
    3. Swap phrases[phraseIndex] and phrases[phraseIndex+1] in the flat words array
    4. Rebuild phrases via buildSessionPhrases
