## Deviations from Plan

### Auto-fixed Issues

**1. pyannote-audio version upgrade 3.3.2 → 4.0.4**
- **Issue:** pyannote 3.3.2 internally passes deprecated use_auth_token to huggingface_hub
- **Fix:** Upgraded to 4.0.4, changed to token= param, handled DiarizeOutput API change
- **Files modified:** scripts/diarize.py, justfile

**2. torchcodec audio decoding failure**
- **Issue:** pyannote 4.x uses torchcodec which is broken with brew ffmpeg on macOS
- **Fix:** Extract audio to WAV via ffmpeg, load with torchaudio, pass waveform dict to pipeline
- **Files modified:** scripts/diarize.py
