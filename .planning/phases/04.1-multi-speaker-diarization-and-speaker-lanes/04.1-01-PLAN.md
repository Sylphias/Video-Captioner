**scripts/diarize.py:**
Create a new Python script that:
1. Accepts CLI args: `<audio_path> <transcript_path> [hf_token]`
   - `audio_path`: path to normalized.mp4
   - `transcript_path`: path to transcript.json (reads existing, writes back enriched)
   - `hf_token`: HuggingFace token (required; error if missing)
2. Emits JSON-line progress to stdout (same protocol as transcribe.py):
   - `{"type": "progress", "percent": 0, "status": "loading_model"}` — on start
   - `{"type": "progress", "percent": 0, "status": "diarizing"}` — after model loaded
   - `{"type": "done", "percent": 100, "speaker_count": N}` — on completion
3. Loads the existing transcript.json, extracts the words array.
4. Runs pyannote diarization:
   ```python
   from pyannote.audio import Pipeline
   import torch
   pipeline = Pipeline.from_pretrained(
       "pyannote/speaker-diarization-3.1",
       use_auth_token=hf_token  # use_auth_token works in 3.3.2
   )
   pipeline.to(torch.device("cpu"))  # NEVER use MPS — known accuracy regression
   diarization = pipeline(audio_path)
   ```
5. Collects segments: `[(turn.start, turn.end, speaker) for turn, _, speaker in diarization.itertracks(yield_label=True)]`
6. Assigns speakers to words using the max-overlap algorithm (cumulative overlap per speaker, not first-match):
   ```python
   for word in words:
       overlap_by_speaker = {}
       for seg_start, seg_end, speaker in segments:
           overlap = min(word["end"], seg_end) - max(word["start"], seg_start)
           if overlap > 0:
               overlap_by_speaker[speaker] = overlap_by_speaker.get(speaker, 0) + overlap
       if overlap_by_speaker:
           word["speaker"] = max(overlap_by_speaker, key=overlap_by_speaker.get)
       else:
           word["speaker"] = None
   ```
7. Writes enriched transcript back to the same transcript.json path (overwrite in place). The transcript gains no new top-level fields — the words array is updated with speaker keys.
8. Error handling: if hf_token is missing, emit `{"type": "error", "message": "HUGGINGFACE_TOKEN required..."}` and sys.exit(1). Wrap the pipeline call in try/except and emit error JSON on failure.
